#### This project features a Jenkins pipeline script designed for building, testing, and deploying a FastAPI application. It integrates with SonarQube for comprehensive code analysis and utilizes Docker for containerization, ensuring a streamlined development process. The deployment is managed through ArgoCD, facilitating continuous delivery and infrastructure as code
